Prompt for Summary

You can customize the prompt for summary.

Title: "{{Title}}"

Transcript: "{{Transcript}}"


Example Prompts:

  • What's key takeaways from the following?
  • Extract the gist of the following.
  • Summarize the following for a 6th grade audience.

Follow-up Prompts

You can set up (up to 3) follow-up prompts. These follow-up prompts will be shown near the prompt inbox.

Strategy for Longer Videos/Articles

ChatGPT has a word limit. For longer videos, the quality of the summary will depend on which part of the video you use. If the content is within the limit, we use the entire content to ensure a comprehensive summary so no worries!
